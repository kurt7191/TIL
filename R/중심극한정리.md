# 중심극한정리 살펴보기



> 표본크기에 따른 표본분포 모양을 살펴보기



### 주사위 함수 생성

```R
dist <- function(n){
  vect <- c()
  for(i in 1:10000){
    tmp <- mean(sample(1:6, n, replace = T))
    vect = append(vect,  tmp)
  }
  return(vect)
}

barplot(table(dist(1)))

```
